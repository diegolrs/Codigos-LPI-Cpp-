1) Sobre o Paradigma de Programação Orientada a objetos, responda as seguintes questões:

(a) O que são classes e objetos? Explique a diferença entre os dois.

Classe é uma estrutura que define o comportamento de objetos através de métodos.
Objeto é uma instância de uma classe. Ele lida com informações recebidas da forma como a classe ordena.
A classe se diferencia do objeto pois é ela quem o manipula e controla suas entradas e saídas.

(b) Para que serve um construtor e quando e como ele pode ser utilizado?

Servem para criar um novo objeto de uma classe determinada.
É utilizado sempre que for preciso um novo objeto para alguma finalidade específica.
Pode ser utilizado com parâmetros vazios aguardando receber argumentos(aguardando dados do get e set), ou
já ser inicializado com parâmetros.

(c) Quais são os benefícios do encapsulamento? Exemplifique.

Independência dos outros objetos e mais proteção, pos os atributos só são.
acessados através de métodos.

Exemplo:
 classe Carro{
private:
double velocidade;
public:
void AlterarVelocidade(double velocidade);
};

void Carro::AlterarVelocidade(double velocidade){
this->velocidade = velocidade;
}

A velocidade do objeto só é alterada pelo método AlterarVelocidade();
